## GUI Installation

Install the `.deb` , `.rpm` or `appimage` file

<Tabs>
    <Tab title="For Ubuntu/Debian">
        ```bash
        sudo apt install ./zetarya.xx.xx.deb
        ```
    </Tab>
    <Tab title="For RHEL/CentOS">
        ```bash
        sudo dnf install ./zetarya.xx.xx.rpm
        ```
    </Tab>
    <Tab title="Linux">
        ```bash
        sudo chmod +x ./zetarya.xx.xx.AppImage
        ./zetarya.xx.xx.AppImage
        ```

        You can keep appimage in the application drawer by writing own script.
    </Tab>
</Tabs>
